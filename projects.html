Wi-Fi Deauther + Wi-Fi Duck
Combined hardware toolkit for Wi-Fi DoS testing and HID injection
Research

A dual-purpose device combining a Wi-Fi deauthentication tool and a USB HID injection platform ("Wi-Fi Duck"), built using a Pro Micro (ATmega32U4) and D1 Mini (ESP8266). The OLED display provides real-time feedback; an RGB LED indicates device state. Powered by a TP4056 Li-ion charging circuit for portability.

Wi-Fi Deauther

Exploits the 802.11 deauthentication frame vulnerability to disconnect devices from a target Wi-Fi network. Used for DoS demonstration and network security auditing. For educational/authorized use only.

Wi-Fi Duck (HID Injection)

When plugged into a target via USB, emulates a keyboard/mouse (HID device). Remote scripts are delivered over the ESP8266 hotspot, allowing keystroke injection undetected by most antivirus software. Useful for security awareness demonstrations.

Smart Home — Multi-Room Automation
4 zones: Bedroom 1 & 2, Living Room, Kitchen
DIY

Room 1 — Bedroom

8-channel relay + NodeMCU ESP8266 controlling tube lights, ceiling fan, Philips Wiz bulb, DIY RGB lamp, study lamp, and 3 sockets. Sockets auto-shutoff at 100% charge to prevent overcharging. AC controlled via Homemate Wi-Fi IR remote with scheduled temperature automation (24°C → 26°C → off by 5 AM).

Room 2 — Bedroom

Philips Wiz smart bulb used as a night lamp (10% brightness). AC controlled via a 40A SSR + NodeMCU ESP8266 installed inside the AC stabilizer, flashed with Tasmota. AC auto-shutoff at 5 AM, stabilizer powers down 5 minutes later. DIY SSR using LED + LDR for optoisolation.

Living Room

Echo Dot 3rd Gen for voice control. 4-channel relay + ESP8266 manages TV, TV bias lighting (LED strip), and 5.1 soundbar. Integrated via Sinric Pro for Alexa/Google compatibility. Custom Alexa routines: "Movie Time" dims lights, enables soundbar via HDMI ARC; "Movie Break" pauses media and restores lights to 50%.

Kitchen

Water pump motor timer using 40A SSR + Arduino Nano. 3-button interface: 2-min, 7-min, and reset. Prevents tank overflow and saves energy. Housed in a standard 2-module electrical box, wired to the existing motor switch. Wi-Fi upgradeable via ESP8266.

Bike Keyless Entry via IoT
Wi-Fi & RF-based proximity keyless system for a motorcycle
DIY

A 4-channel relay + NodeMCU ESP8266 connected to the bike's ignition, kill switch, and self-start — all controllable via Alexa, Google Assistant, or a WearOS app. Powered by a 12V-to-USB adapter from the bike battery. An RF module salvaged from an RC car was converted into a keychain remote for proximity-based auto-lock/unlock (locks beyond ~10 m). Mobile hotspot with matching SSID bridges the home network while on the move. A DIY Bluetooth receiver in the helmet enables helmet-triggered engine start.

  • Ignition, kill switch, and self-start via app/voice
  • Proximity auto-lock/unlock via salvaged RF module
  • Helmet-triggered start via Bluetooth receiver
  • WearOS Alexa app integration

Villa Network — Multi-Compound Wi-Fi Deployment & NOC Dashboard
35-room, 160+ concurrent user network across two compounds · Freelance · Dec 2024
Live NOC Dashboard
Freelance

Designed and deployed a cost-effective enterprise-grade Wi-Fi network across two villa compounds with 35 rooms, supporting 110+ Wi-Fi users and 160+ peak concurrent users. All 16 routers were repurposed from existing hardware and configured as access points with static IPs and non-overlapping channels to minimise interference. Achieved 200–230 Mb/s stable throughput — all monitoring infrastructure built at zero additional cost.
